HTML

Hypertext Markup Languge (HTML) is the standard Markup languge for creationg web pages and web applications. web browsers receive HTML documents from a web server or HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included for the appearance of the document.

HTML elements are the building blocks of HTML pages. Whith HTML constructs, images form, may be embedded into the rendered page.
